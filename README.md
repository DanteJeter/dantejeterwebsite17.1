# Dante Jeter Website — Version 15.1

Version 15 preserves the approved Version 14.2 design and expands it into a scalable real estate platform.

## Core systems
- Manually selected three-property homepage showcase
- Listing lifecycle statuses: Active, Coming Soon, Price Improvement, Under Contract, Pending, and Sold
- Gold Under Contract image ribbons while property pages remain live
- Recently Sold results gallery
- Related listings on property showcase pages
- Marketing Center
- Seller Success Hub
- Community Spotlight
- Market Insights
- Existing lead capture and compliance disclosures preserved

Edit `listings-config.js` to add verified listing information, set `featured: true`, assign `featuredOrder`, or change a lifecycle status.
