# Dante Jeter Digital Platform

This repository is the reusable foundation for listing showcases, community pages, seller valuation, buyer/seller resources, SEO content, analytics, and lead management. Every release should improve the shared platform rather than create an isolated microsite.

## Current
- Reusable business homepage and property showcase
- Master gallery order and launch checklist
- Google Sheets lead logging, email notification, tracking, and branded thank-you flow

## Backlog
- Secure BoldTrail API service
- AI description generator
- Community pages and market reports
- Lead dashboard and admin panel
- Unique property domains
- Expanded reusable component system

## Release naming
Use `Dante Jeter Digital Platform vX — [Primary Feature or Property]`.
