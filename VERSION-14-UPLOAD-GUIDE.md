# Version 14 Upload Guide

Upload the **contents** of this folder to the root of the GitHub repository.

The exact property media folder is:

```text
assets/104-paul-avenue/
```

Do not rename this folder and do not create `media` or `gallery` subfolders.

Required media files include:

- `hero.jpg`
- `photo-01.jpg` through `photo-17.jpg`
- `video-poster.jpg`
- `walkthrough.mp4`

After Vercel deploys, open `/media-check.html` and run the media check. Every item should report **OK**.
