# Version 16 Listing System

## Add a future listing
1. Create `assets/<property-slug>/`.
2. Add `hero.jpg` and numbered gallery photos.
3. Duplicate `property-333-patricia-config.js`, update the property fields and photo list.
4. Duplicate `333-patricia-avenue.html`, change the config script reference and verified property copy.
5. Add one record to `listings-config.js`. Homepage and listings hub cards update automatically.

Never publish unverified price, square footage, MLS, taxes, school, or year-built data.
